# Python-project
Проект реализуется в формате Telegram-бота, функционирующего как цифровой историко
научный календарь с элементами просветительского контента.
Результат будет представлен в виде Telegram-бота «Научный календарь», который еже
дневно отправляет подписчикам «Событие дня»– мультимедийную карточку, содержа
щую информацию о значимом событии в истории науки или дне рождения учёного.
